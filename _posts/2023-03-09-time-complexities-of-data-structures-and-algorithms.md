Here we only focus on average time complexity when you are searching or inserting with certain collection. 


| Data Structure | search ATC| insert ATC |
| -------- | -------- | -------- |
| Array     | O(N)     |  O(N)    |
| -------- | -------- | -------- |
| List     | O(N)     |   O(1)    |
| -------- | -------- | -------- |
| Queue     | O(N)     |   O(1)    |
| -------- | -------- | -------- |
| Stack     | O(N)     |    O(1)   |
| -------- | -------- | -------- |
| Hash Map     | O(1)     |  O(1)     |
| -------- | -------- | -------- |
| Binary Tree     | O(log N)     |  O(log N)    |
